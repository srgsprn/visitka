# Деплой бота на сервер (работа 24/7)

Пока бот запущен на твоём ноутбуке — он работает. Закрыл ноутбук или выключил — бот перестаёт отвечать. Чтобы он работал **автономно**, его нужно развернуть на облачном сервисе.

Ниже — два простых варианта: **Railway** и **Render**. Оба дают бесплатный или недорогой тариф и подходят для этого бота.

---

## Вариант 1: Railway (проще всего)

1. Зайди на [railway.app](https://railway.app), войди через GitHub.
2. **New Project** → **Deploy from GitHub repo** → выбери репозиторий с ботом (или залей папку `telegram-crop-bot` в отдельный репо).
3. Укажи **Root Directory**: `telegram-crop-bot` (если бот лежит в подпапке репо).
4. **Variables** → добавь переменную:
   - `BOT_TOKEN` = твой токен от BotFather.
5. **Settings** → **Deploy**:
   - **Start Command**: `python bot.py`  
   Или оставь пустым, если используешь Procfile с `worker: python bot.py`.
6. Деплой запустится. Первый запуск может занять 2–3 минуты (скачивание модели rembg).

Готово: бот будет работать постоянно, пока проект на Railway запущен.

---

## Вариант 2: Render

1. Зайди на [render.com](https://render.com), войди через GitHub.
2. **New** → **Background Worker**.
3. Подключи репозиторий. **Root Directory**: `telegram-crop-bot` (если бот в подпапке).
4. **Build Command**: `pip install -r requirements.txt`  
   **Start Command**: `python bot.py`
5. **Environment** → добавь переменную `BOT_TOKEN` со значением токена.
6. **Create Background Worker**.

На бесплатном тарифе воркер может «засыпать» после периода неактивности; при первом сообщении боту может понадобиться 30–60 секунд на пробуждение. Платный тариф держит бота всегда включённым.

---

## Про Vercel

**Vercel** рассчитан на короткие HTTP-запросы (серверлесс), а не на постоянно работающий процесс. Чтобы бот работал на Vercel, нужно переводить его на **webhooks** (Telegram шлёт запросы на твой URL). Плюс rembg тяжёлый (модель ~176 МБ, много памяти и времени на «холодный» старт), из‑за чего на бесплатном тарифе Vercel возможны таймауты и лимиты памяти. Для этого бота удобнее **Railway** или **Render**.

## Кратко

| Где запущен        | Работает когда ноутбук выключен? |
|--------------------|-----------------------------------|
| Твой ноутбук       | Нет                               |
| Railway / Render   | Да, 24/7 (или с «сном» на Render free) |

Для автономной работы залей бота в GitHub и разверни на **Railway** или **Render** по шагам выше.
