# Готовая визитка — как открыть

1. Замените в `index.html` имя, профессию, описание, текст «О себе» и контакты на свои.
2. Замените фото: положите своё изображение в эту папку и назовите его `portrait.jpg` (или измените в HTML путь к картинке).
3. Откройте сайт в браузере:
   - **macOS:** дважды кликните по `index.html` или в терминале: `open index.html`
   - **Windows:** дважды кликните по `index.html`
   - Или перетащите `index.html` в окно браузера.

Файлы: один HTML (`index.html`) и одно фото (`portrait.jpg`). Внешние библиотеки не нужны.
